2. PowerShell Version Compatibility (‚ö†Ô∏è CRITICAL)ÔÉÅ

Issue: Script must maintain PowerShell 5.1 compatibility Discovery: Many organizations still on PS5.1; PS7 features break compatibility Evidence:

  • No ?? null coalescing operator in PS5.1

  • No ForEach-Object -Parallel in PS5.1

  • Different module loading behavior Resolution: Avoid PS7-only syntax; use ThreadJob module for parallelization Critical Learning: Always test on PS5.1 before deployment

6. Development Environment Setup RequirementsÔÉÅ

Required Software:

  • PowerShell: 5.1+ (PS5.1 compatibility required)

  • Unity: 2021.1.14f1 (.NET Standard 2.0)

  • Claude CLI: v1.0.53+ or API key

  • Memory: 4GB minimum

  • Storage: 1GB for logs/database

Required Modules:

Environment Variables:

$env:PSModulePath = "$PWD\Modules;$env:PSModulePath"
